# LivingContext Documentation (public)

Public, customer-facing documentation for [LivingContext](https://livingcontext.ai), in [Mintlify](https://mintlify.com) format.

## Structure

- `docs.json` — Mintlify site configuration and navigation
- `index.mdx` — landing page
- `getting-started/`, `guides/`, `agents/`, `capture/`, `integrations/`, `admin/` — documentation pages (MDX)

## Local preview

```bash
npm i -g mint
mint dev
```

Then open http://localhost:3000.
